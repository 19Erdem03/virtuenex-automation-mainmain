# VirtueNex Automation - Backend & Dashboards

Welcome to the **VirtueNex Automation** project repository! This project serves as the core backend, CRM, and client/admin dashboard infrastructure tailored specifically for an AI Automation Agency focused on Real Estate professionals.

## ðŸš€ Overview

The system provides robust authentication, database management, and interactive dashboards to handle:
- **Client System Deployments**: Management of AI Chat Agents, Inbound Phone Agents, and Data Sync integrations.
- **Lead Capture & CRM**: Automated capture and management of leads generated by AI agents for real estate clients.
- **Roles & Permissions**: Distinct access levels for **Admins** (VirtueNex staff), **Clients** (Real Estate Agents/Brokers), and **Leads** (Prospective Buyers).
- **Property Listings & Tour Bookings**: Showcase properties, set availability, and allow leads to book/manage property tours.
- **Crypto Payments**: Manual verification workflow for crypto transactions.

## ðŸ›  Tech Stack

This project is built with a modern, scalable web stack:

### Frontend
- **Framework**: [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router v6](https://reactrouter.com/)

### Backend & Database
- **Platform**: [Supabase](https://supabase.com/)
- **Database**: PostgreSQL (with Row Level Security for data privacy)
- **Authentication**: Supabase Auth

### Monitoring & Error Tracking
- **Application Monitoring**: [Sentry](https://sentry.io/) (`@sentry/react` for the frontend)

## ðŸ’» Getting Started for Developers

### Prerequisites
- Node.js (v18+ recommended)
- npm
- Access to the Supabase project configuration

### Local Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Environment Variables:**
   Ensure you have a `.env` file at the root of the project with your Supabase credentials, Sentry DSN, and other necessary variables.
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Type Checking & Linting:**
   - Run `npm run typecheck` to verify TypeScript types.
   - Run `npm run lint` to enforce code quality.
